# 4-DOF-Manipulator-C-A-CC
This is a project that implements a 4 DOF Manipulator in Minecraft using **Create Aeronautics** and **CC: Tweaked**.

**Previews**: https://imgur.com/a/4-dof-aeronautics-arm-test-BfevJpu

![Demo](./demo.gif)

### Installation
To download, copy the following commands into your Minecraft computer terminal: 

```bash
wget https://raw.githubusercontent.com/CraftxTD/6-DOF-Manipulator-C-A-CC/refs/heads/main/startup.lua  /startup.lua
```

then run the following script in the root directory:

```bash
startup
```
